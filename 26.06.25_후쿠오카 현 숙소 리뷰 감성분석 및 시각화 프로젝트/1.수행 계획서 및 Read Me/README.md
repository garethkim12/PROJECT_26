# 후쿠오카 현 숙박업소 리뷰 감성분석 프로젝트

> 일본 후쿠오카 현 숙박업소(호텔·료칸·게스트하우스) 이용자 리뷰 81,613건에 대한  
> 한국어 자연어처리 기반 감성분석 및 시각화 대시보드 프로젝트

---

## 프로젝트 개요

| 항목 | 내용 |
|---|---|
| **데이터 출처** | Google Maps / Tripadvisor (크롤링) |
| **분석 대상** | 후쿠오카 현 숙박업소 리뷰 (한국어) |
| **최종 분석 건수** | 81,613건 (204개 업소, 3개 카테고리) |
| **분석 기간** | 2008년 ~ 2026년 6월 |
| **감성 분류** | 긍정 / 부정 (2분류) |
| **최종 모델 F1** | 0.8691 (베이스라인 대비 +17.5%) |

---

## 주요 분석 결과

### 감성 분포
- 긍정 71.9% (58,640건) / 부정 28.1% (22,973건)
- 평균 평점 4.07 / 5.0

### RQ1. 평점-감성 불일치
- 5점인데 부정 리뷰: **3,777건(4.6%)** — "별점은 후하게, 텍스트엔 불만"
- 1점인데 긍정 리뷰: **325건(0.4%)** — 역접 표현 패턴

### RQ2. 숙소 유형별 비교
| 유형 | 건수 | 평균 평점 | 부정률 |
|---|---|---|---|
| 호텔 | 71,994 | 4.06 | 28.6% |
| 료칸 | 6,738 | 4.19 | 23.6% |
| 게스트하우스 | 2,881 | 4.10 | 26.9% |

### RQ3. 시계열 트렌드
- 코로나 이전(2017~2019) 평균 긍정률: 85.8%
- 코로나 기간(2020~2021) 평균 긍정률: 86.2% *(리뷰 수 감소, 만족 고객 위주 남음)*
- 코로나 이후(2022~2026) 평균 긍정률: 69.3% *(여행 재개 후 다양한 경험 반영)*

## 프로젝트 구조

```
sentiment_project/
├── app.py                      # Streamlit 대시보드 (RQ1~RQ4)
├── preprocess.py               # 전처리 모듈
├── run_pipeline.py             # 감성분석 실행 스크립트
├── validate_model.py           # 5종 모델 비교 검증
├── 파인튜닝_v2_Colab.ipynb     # 파인튜닝 노트북 (의사레이블 포함)
├── 대시보드_Colab실행.ipynb    # Streamlit Colab 실행 노트북
├── labeling_sample.csv         # 직접 레이블링 샘플 500건
├── full_cleaned.csv            # 전처리 완료 데이터 (81,613건)
├── final_result_v3.csv         # 최종 감성분석 결과
└── requirements.txt            # 패키지 목록
```

---

## 실행 방법

### 1. 환경 설정
```bash
pip install -r requirements.txt
```

### 2. Streamlit 대시보드 실행 (로컬)
```bash
streamlit run app.py
```
사이드바에서 `final_result_v3.csv` 업로드 후 사용

### 3. Colab에서 실행
`대시보드_Colab실행.ipynb`를 Colab에서 열어 순서대로 실행
