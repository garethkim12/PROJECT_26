# EVASION — 생존 프로토콜 (v9)

사방에서 날아오는 장애물을 피해 최대한 오래 생존하는 2D 캔버스 아케이드 게임입니다.
HTML5 Canvas와 순수 JavaScript만으로 제작되었으며, 별도 설치나 빌드 과정 없이 브라우저에서 바로 실행됩니다.

1. `EVASION_v9.html` 파일을 더블클릭하면 기본 브라우저(Chrome, Edge 등)에서 바로 열립니다.

## 🧩 핵심 게임 시스템

### 기본 규칙
- 화면 사방(상하좌우)에서 장애물이 날아오며, 플레이어를 향해 약하게 유도됩니다.
- 장애물에 닿으면 라이프 1개가 감소하고, 약 1.8초간 무적 상태가 됩니다.
- 라이프가 0이 되면 게임 오버, 생존 시간에 비례해 점수가 누적됩니다.
- 시간이 지날수록 레벨이 올라가며 장애물의 속도·수량·유도성이 강해집니다.

### 난이도 (시작 화면에서 선택)

| 난이도 | 라이프 | 속도 배율 | 스폰 배율 | 레벨업 주기 | 점수 배율 |
|---|---|---|---|---|---|
| EASY (쉬움) | 5개 | ×0.70 | ×0.65 | 16초 | ×0.7 |
| HARD (어려움) | 5개 | ×1.00 | ×1.00 | 12초 | ×1.0 |
| EXTREME (익스트림) | 3개 | ×1.40 | ×1.45 | 8초 | ×1.5 (+서지 보너스) |

EXTREME 난이도는 50초 생존 시 전체 수치가 추가로 ×1.5 강화되는 **서지(SURGE)**가 발동합니다.

### 특수 이벤트 (10초마다 랜덤 발생)

| 이벤트 | 설명 |
|---|---|
| **BARRAGE** | 화면 한 면에서 20개 이상의 파편이 일렬로 연속 발사 |
| **TRIPLE METEOR** | 거대 운석 3개가 서로 다른 방향에서 무작위 궤도로 진입 |
| **SCATTER8** | 8방향에서 동시에 파편 다수가 날아옴 |
| **MULTI LASER** | 무작위 방향(수평/수직)의 레이저 3줄기가 2초간 동시 발사 |

이벤트 발생 전 2.5초간 화면 상단에 경고 배너가 표시됩니다.

### 감마 레이저 (보너스 기믹)
- 70초 생존 후부터 20초마다 발동합니다.
- 수평 또는 수직 방향의 레이저가 경고(점선) 후 2초간 발사됩니다.

### 리더보드
- 플레이어 이름, 점수, 생존 시간, 난이도가 함께 기록됩니다.
- 상위 10위까지 자동 정렬되며, 브라우저의 `localStorage`에 저장됩니다.
- 같은 브라우저에서 재방문 시 기록이 유지됩니다. (다른 브라우저/기기와는 공유되지 않습니다)

---

## ⚠️ 알아두어야 할 점

- 현재 리더보드는 **로컬 저장(localStorage)** 방식으로, 플레이어 개인의 브라우저에만 기록이 저장됩니다.
- 여러 사용자가 기록을 공유하는 **글로벌 리더보드**를 원하실 경우 별도의 백엔드(Firebase 등) 연동이 필요합니다.
- 외부 배포(Netlify, GitHub Pages 등)를 원하실 경우 `EVASION_v9.html` 파일 하나만 업로드하면 즉시 작동합니다.
